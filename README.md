Project Name: Mini Shopping Website
Description
-A mini shopping website to specifically buy 1 item only.

Installation Instructions
git clone https://github.com/computerririn-coder/Buy-Your-Own-Nene
cd project
npm install
npm start

Usage

Enter the site.
-Click on any option in the "quality options". There's a "cuteness option" but that is optional, and you can proceed to click the purchase button without choosing any option in the "cuteness options".

Technologies Used
-HTML, CSS, JavaScript, React, Git
